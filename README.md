# 🐙 Coding Mascot Generator

An AI-powered app that generates a custom mascot character for coding tutorials and dev brands, built with **Google AI Studio** using **Gemini** for personality writing and **Imagen** for artwork.

[Live app →](https://cute-coding-mascot-generator-924052444918.us-east1.run.app)

## The Prompt

> "Please create an app that generates a cute mascot character for a coding/tutorial brand, using Imagen for the visuals and Gemini to create a name and short personality description for the mascot. The user should be able to type in a few style keywords (like 'friendly owl', 'cool robot', 'cheerful fox') and get a unique mascot image along with its name and bio."

## What It Built

A few prompts in, Gemini turned this into **MascotCraft Studio** — a full app with:

- A "Character Designer" with quick preset ideas (Wise Python Owl, Cyberpunk JS Fox, Debugging Robo Kitty, Coffee Coding Monkey, Chubby Bash Sloth, Little Go Gopher)
- Color palette options (Cyberpunk Glow, Forest Sage, Nordic Frost, Solar Ember, Cosmic Violet)
- Multiple visual rendering styles (3D Chibi Toy, Minimal Vector, 16-Bit Retro Pixel, Circular Badge)
- A "Studio Gallery Showcase" that saves generated mascots locally so you can revisit them
